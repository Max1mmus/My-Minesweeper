# My-Minesweeper
My first minesweeper game

### After you clone repo :

1. run `npm install` to install packages.
2. run `npm run open` to start local serv. @ http://localhost:1234/

 ## Made with:
 
 * Pure JavaScript
 * CSS
 * HTML
